Temperature Converter Project

About This Project

This project is a simple temperature converter. The user will input a temperature in either Fahrenheit or Celsius and press a "Convert" button. The converted temperature will then be displayed with the correct unit.

It's a great project for practicing basic form handling, input validation, and simple logic in JavaScript, along with HTML and CSS for the interface.

UI Requirements-
 UI should include four main elements:


Input Field – Lets the user type in their temperature. Making  sure to validate that the input is actually a number before doing any conversion.
Dropdown Menu or Radio Buttons – Lets the user choose whether they're inputting in Celsius or Fahrenheit.
Convert Button – Triggers the temperature conversion when clicked.
Display Area – Shows the final converted temperature along with the correct unit.


Extra Challenge

For an added challenge,  i can include a third conversion option: Kelvin. This means users should be able to convert to and from Kelvin as well, in addition to Celsius and Fahrenheit.

What I 'll Practice


Handling user input and validating it (checking for non-numeric values)
Working with dropdowns/radio buttons to capture user selection
Writing conversion logic in JavaScript
Updating the DOM dynamically to display results
Basic event handling with buttons


Goal

By the end of this project, I should have a working temperature converter where users can input a value, pick a unit, hit convert, and instantly see the correct converted result — with bonus support for Kelvin if  I CAN  take on the extra challenge.
